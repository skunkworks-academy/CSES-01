# Capstone Scenario

## Scenario

A customer is running multiple Azure workloads with inconsistent security configuration. They also plan to launch an AI service that uses internal business data. The customer wants a practical 30-day plan that improves visibility, reduces risk, and creates an operational security rhythm.

## Required deliverables

Produce:

1. Executive risk summary.
2. Top five security posture findings.
3. Defender for Cloud configuration recommendations.
4. Workload protection selection matrix.
5. AI security control checklist.
6. 30-day remediation plan.
