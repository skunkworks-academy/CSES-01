### 1. What is the primary purpose of Cloud Security Posture Management?

A. To replace all security operations workflows  
B. To identify, prioritize, and guide remediation of cloud security risks  
C. To deploy applications automatically  
D. To manage billing and subscriptions only  

**Answer:** B

### 2. Which item is most directly associated with reducing identity blast radius?

A. Increasing storage capacity  
B. Assigning broad Contributor permissions  
C. Least privilege RBAC and scoped role assignments  
D. Disabling all logging  

**Answer:** C

### 3. What is a key reason to enable workload protection plans?

A. To provide threat detection and workload-specific protection  
B. To remove the need for governance  
C. To replace all Azure policies  
D. To create users automatically  

**Answer:** A

### 4. Which risk is specific to AI-enabled workloads?

A. Prompt injection  
B. Incorrect DNS TTL  
C. Low disk space  
D. Expired SSL certificate only  

**Answer:** A

## Section B â€” Scenario questions

### Scenario 1

A customer has many high-severity recommendations but limited engineering capacity. Explain how you would prioritize the first five remediation actions.

**Expected points:**

- Business criticality
- Internet exposure
- Identity privilege
- Sensitive data
- Exploitability
- Compliance impact
- Remediation effort

### Scenario 2

A new AI workload uses a managed identity with broad Contributor rights over the subscription. What risks does this create and how should it be remediated?

**Expected points:**

- Excessive privilege
- Blast radius
- Scope identity to required resources
- Replace broad RBAC with least privilege
- Review Key Vault and data access
- Monitor with Defender for Cloud
