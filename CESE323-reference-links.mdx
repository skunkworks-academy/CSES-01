## 13. Suggested Learner Workflow

1. **Start with the CESE323 Microsoft Learn Collection.**  
   Use it as the main guided learning route.

2. **Complete pre-work if needed.**  
   Learners who are new to Microsoft security should start with SC-900 and the security, compliance, and identity fundamentals content.

3. **Review the Defender for Cloud docs before labs.**  
   Confirm portal access, subscription scope, RBAC, Defender plan availability, and cost approval.

4. **Use workload-specific documentation during hands-on labs.**  
   Select the correct Defender plan references for Storage, Containers, App Service, AI Services, or Foundry-based scenarios.

5. **Use architecture frameworks during strategy discussions.**  
   Map technical findings to Zero Trust, CAF, WAF, MCRA, and MCSB.

6. **Check release notes before delivery.**  
   Defender for Cloud, Microsoft Purview, Microsoft Entra, and AI security features change regularly.
